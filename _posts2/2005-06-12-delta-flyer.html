วิธีการแก้โค้ด Html

ต้องการจะแก้โค้ด Html บนบล๊อค ทำอย่างงี้น๊า.....Edit It!

1) ไปที่ Link http://www.siteexperts.com/blogging/editit.htm คลิ้กขวาที่ Edit it แร้วกด Add to Favorite ...จะมี Security Popup ขึ้นมา ให้กด yes
2) ไปที่หน้า add blog entry
3) เข้า favorite ไปเปิด link ที่ add เมื่อกี้เรย (Edit it อ่ะค่ะ)
4) จะเห็น ปุ่ม Use HTML to creat your page. โผล่ขึ้นมา บริเวณ เหนือปุ่ม Add photos น๊า
5) ให้ ติ้กเครื่องหมายถูกลง ไป ในช่องสี่เหลี่ยม ...ทีนี้ ใน blog คุณจะมี code โผล่ขึ้นมา (แนะนำว่าให้เขียนอะไรใน blog ให้เสร็จก่อนนะ)
